Frequently Asked Questions

Do credits expire?

No — your credits never expire. Buy once, use whenever you want.

What's the difference between one-time, monthly, and annual?

One-time: buy a credit pack once, no recurring charges. Monthly: auto-renews every month for convenience. Annual: auto-renews yearly with 20% savings vs monthly.

Do I need my own Anthropic API key?

No — CodeSail handles all API calls on our end. Just login with GitHub and start using it.

What AI model does CodeSail use?

We use Claude Sonnet 4.6 by Anthropic — one of the best models for code understanding and generation.

Is my code sent to your servers?

Only the code context you're actively working on is sent to process your request. We don't store your code.

What payment methods do you accept?

India: UPI, cards, net banking, wallets via Razorpay. Global: coming soon.
