Frequently Asked Questions

Do I need my own Anthropic API key?

No — CodeSail handles all API calls on our end. Just login with GitHub and start using it immediately.

What AI model does CodeSail use?

We use Claude Sonnet 4.6 by Anthropic — one of the best models for code understanding and generation.

Can I cancel anytime?

Yes — cancel from inside VS Code anytime. No questions asked. You keep access until the end of your billing period.

Is my code sent to your servers?

Only the code context you're actively working on is sent to process your request. We don't store your code.

What payment methods do you accept?

We accept all major credit/debit cards, UPI, net banking, and wallets via Razorpay.
